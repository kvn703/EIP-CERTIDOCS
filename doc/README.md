# Documentation Site Guide

Ce dossier contient le code source du site de documentation du projet. C'est une application React construite avec Vite.

## Prérequis

- Node.js installé sur votre machine.
- npm (généralement inclus avec Node.js).

## Installation

Avant de lancer le projet, assurez-vous d'installer les dépendances :

```bash
npm install
```

## Lancer le serveur de développement

Pour démarrer le site en local et voir les modifications en temps réel :

```bash
npm run dev
```

Le terminal affichera l'URL locale (généralement `http://localhost:5173/`) sur laquelle vous pouvez consulter la documentation.

## Construction pour la production

Pour générer les fichiers statiques optimisés pour la production :

```bash
npm run build
```

Les fichiers générés se trouveront dans le dossier `dist`.

## Prévisualiser la production

Pour tester le build de production localement :

```bash
npm run preview
```
